# Snowflake Inspector Expert

Use this skill when reviewing `snowflake-inspector` output.

Findings are written to `~/.cache/claude-grc/findings/snowflake-inspector/<run_id>.json`.

Key mappings: `IAC-02` MFA, `NET-06` network policies, `DCH-06` masking, `DCH-07` row access, `IAC-04` session timeout, `DCH-09` retention.
