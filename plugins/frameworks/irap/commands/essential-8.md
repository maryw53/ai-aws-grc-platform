# Essential Eight Strategies

The Australian Cyber Security Centre's Essential Eight mitigation strategies to prevent cyber security incidents.

## 8 Strategies

1. **Application Control**: Prevent execution of unapproved applications
2. **Patch Applications**: Update applications within 48 hours (critical)
3. **Configure Macro Settings**: Disable macros from internet
4. **User Application Hardening**: Disable unnecessary features
5. **Restrict Admin Privileges**: Least privilege access
6. **Patch Operating Systems**: Update OS within 48 hours (critical)
7. **Multi-Factor Authentication**: MFA for all users
8. **Regular Backups**: Daily backups with offline storage

## Maturity Levels

- **Level 1**: Basic implementation
- **Level 2**: Standard implementation
- **Level 3**: Advanced implementation
