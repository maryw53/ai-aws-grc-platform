# Classification Level Selection

Determine appropriate Australian government security classification.

## Classification Levels

- **OFFICIAL**: Routine government business
- **OFFICIAL:Sensitive**: Personal information
- **PROTECTED**: Cabinet, national security
- **SECRET**: Intelligence operations
- **TOP SECRET**: Highest sensitivity

## Control Requirements

Classification determines required security controls from ISM.

## Examples

```bash
/irap:classification
```
