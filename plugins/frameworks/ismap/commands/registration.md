# ISMAP Registration

Process for registering cloud services with the Japanese government ISMAP program.

## Registration Steps

1. ISO 27001/27017/27018 certification
2. Submit application to Digital Agency
3. Technical assessment
4. Registration approval
5. Listing on ISMAP registry

## Timeline

6-12 months for full registration process.

## Examples

```bash
/ismap:registration
```
